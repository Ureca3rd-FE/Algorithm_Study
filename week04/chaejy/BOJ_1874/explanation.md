## ✍️ 풀이 아이디어
- 목표 수열을 왼쪽부터 하나씩 x라 하면 x가 나올 때까지 push
- next ≤ x 라면 next를 스택에 push하고 next++(연산 기록에 +)
- 이걸 스택 top이 x 이상이 될 때까지 반복
- x를 pop해서 맞추기

## 📊 복잡도 분석
- 시간 복잡도
    - O(N) 1부터 N까지의 각 수는 스택에 최대 1번 push, 1번 pop 됩니다. 비교/분기 역시 각 원소당 O(1)이라 총 O(N)
- 공간 복잡도: O(N)
